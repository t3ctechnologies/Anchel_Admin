Technical accounts

This view allows you to create accounts with specific delegation rights. Details of rights:

  • 'Delegation' : enables operations for other users through the Delegation API.To carry out these operations you must select the list of permissions (rights)that we want to give to this account. No permissions are selected by default.
    For the definition of how the Delegation API works, you can refer to its available documentation at the address : http://download.linshare.org
  • 'Upload proposal' : Upload proposal being an external module of the application, it requires a special account to connect to Anchel and recover its configuration. This account is dedicated to this use.
